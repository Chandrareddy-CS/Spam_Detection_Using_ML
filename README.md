# 📧 Spam Detection Using Machine Learning

## 🚀 Overview
This project detects whether a message is **Spam or Not Spam** using a Machine Learning model (**Support Vector Machine - SVM**).  
It also includes a simple **Flask-based web interface** for user interaction.

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- HTML (basic frontend)
- Flask (for web app)

---

## 📊 Dataset
- SMS Spam Collection Dataset  
- Contains labeled messages as spam or ham

---

## 🚀 Features
- Classifies messages as **Spam or Not Spam**
- Machine Learning model using SVM
- Simple web interface for input
- Fast and accurate predictions

---

## ▶️ How to Run

 1. Install dependencies
```bash
pip install pandas scikit-learn flask nltk
```
 2. Run the project
```bash
python Project.py
```
 3. Open in browser
```bash
http://127.0.0.1:5000
```
### 🌐 Web App

This project includes a Flask-based web application where users can:

Enter a message
Get prediction instantly (Spam / Not Spam)

### 📈 Model Performance
Accuracy: 98.57%
Precision: High for both spam and non-spam classes
Recall: Strong detection capability

The model performs well in distinguishing spam and legitimate messages.
