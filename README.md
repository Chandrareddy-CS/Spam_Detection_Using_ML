# 📧 Spam Detection Using Machine Learning

## 🚀 Overview
This project detects whether a message is **Spam or Not Spam** using a Machine Learning model (**Support Vector Machine - SVM**).  
It also includes a simple **Flask-based web interface** for user interaction.

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- HTML (basic frontend)
- Flask (for web app)

---

## 📊 Dataset
- SMS Spam Collection Dataset  
- Contains labeled messages as spam or ham

---

## 🚀 Features
- Classifies messages as **Spam or Not Spam**
- Machine Learning model using SVM
- Simple web interface for input
- Fast and accurate predictions
