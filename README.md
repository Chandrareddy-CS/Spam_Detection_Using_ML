 Spam Detection Using Machine Learning

 Overview
This project detects spam messages using a Machine Learning model (SVM).

 ⚙️ Tech Stack
- Python
- Scikit-learn
- Pandas
- HTML (basic frontend)

📊 Dataset
- SMS Spam Collection Dataset

 🚀 Features
- Classifies messages as Spam or Not Spam
- Simple frontend for user input
- Trained using Support Vector Machine (SVM)

 ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the script:
   python Project.py
